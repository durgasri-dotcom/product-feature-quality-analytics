## Data

Raw data is excluded from version control.

**Source:** [eCommerce behavior data — Kaggle](https://www.kaggle.com/datasets/mkechinov/ecommerce-behavior-data-from-multi-category-store)

**To reproduce:**

1. Download `2019-Oct.csv` from Kaggle
2. Place it in `Desktop/archive/`
3. Run `python blend_data.py`

This generates 1M blended rows in `data/raw/product_logs.csv`.
